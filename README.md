# 語学学習サポートAI

AIに外国語のことを質問すると、やさしく教えてくれて、答えを自動でスプレッドシートにメモしてくれるアプリです。

英語・中国語・韓国語・スペイン語・フランス語・アラビア語に対応しています。

---

## できること

- 外国語の表現や意味について、AIにチャットで質問できる
- 大事なフレーズ・例文・ニュアンスをAIが自動でスプレッドシートに保存してくれる
- フレーズや例文の横にある🔊ボタンで、ネイティブの発音を聞ける
- AIモデルを3つから選べる（Flash Lite / Flash / Pro）
- `Ctrl + Enter` でメッセージを送信できる
- スマホ・タブレット・PCどれでも使える

---

## セットアップ手順

### 用意するもの

- Googleアカウント（Gmailが使えればOK）
- Gemini APIキー（無料で取得できます。取り方はこの下で説明します）

---

### ステップ 1: Gemini APIキーを取得する

APIキーとは、「このアプリからGemini AIを使っていいですよ」という許可証のようなものです。
無料で取得できます。

#### 1-1. Google AI Studio を開く

1. ブラウザで **Google AI Studio**（https://aistudio.google.com/）を開く
2. Googleアカウントでのログインを求められたら、ログインする
   - まだGoogleアカウントを持っていない場合は、先にGmailアカウントを作成してください

#### 1-2. APIキーのページに移動する

1. 画面の左側にメニューが並んでいるので、その中から **「API キー」**（英語表示の場合は **「Get API key」**）をクリック
2. APIキーの管理ページが開く

#### 1-3. APIキーを作成する

1. 画面右上の**「APIキーを作成」**（英語: **「Create API key」**）ボタンをクリック
2. 「Google Cloudプロジェクトを選択してください」という画面が出る場合がある:
   - すでにプロジェクトが表示されていたら → そのまま **「既存のプロジェクトでAPIキーを作成」** をクリック
   - 何もなければ → **「新しいプロジェクトでAPIキーを作成」** をクリック（自動でプロジェクトが作られます）
3. しばらく待つと、APIキーが生成される

#### 1-4. APIキーをコピーして保存する

1. 画面に **長い英数字の文字列**（`AIzaSy...` のような文字列）が表示される。これがAPIキー
2. **「コピー」ボタン** をクリック（またはキーの文字列を選択して `Ctrl + C`）
3. メモ帳やテキストファイルなどに **貼り付けて保存** しておく（あとのステップ6で使います）

> **注意:** APIキーは **絶対に他の人に見せないでください**。パスワードと同じくらい大切なものです。
> SNSやブログに載せたり、友達に送ったりしないようにしましょう。

> **料金について:** Gemini APIには無料枠があり、個人の学習用途であれば無料枠の範囲内で十分に使えます。
> 万が一の課金が心配な場合は、Google Cloud コンソール（https://console.cloud.google.com/）で予算アラートを設定しておくと安心です。

---

### ステップ 2: Googleスプレッドシートを作る

1. **Googleドライブ**（https://drive.google.com/）を開く
2. 左上の **「+ 新規」** ボタンをクリック
3. **「Google スプレッドシート」** → **「空白のスプレッドシート」** をクリック
4. 新しいスプレッドシートが開く（名前はそのままでOK。好きな名前をつけてもOK）

---

### ステップ 3: Apps Script エディタを開く

1. スプレッドシートの上のメニューから **「拡張機能」** をクリック
2. **「Apps Script」** をクリック
3. 新しいタブでApps Scriptエディタが開く

---

### ステップ 4: バックエンドのコードを貼り付ける

1. エディタの真ん中に、最初から `function myFunction() { }` のようなコードが書いてある
2. **そのコードを全部消す**（`Ctrl + A` で全選択 → `Delete` キー）
3. このリポジトリにある **`main.gs`** ファイルの中身を **すべてコピー** する
4. エディタに **貼り付ける**（`Ctrl + V`）
5. 上にあるファイル名「コード.gs」をクリックして、名前を **`main`** に変える（`.gs` は自動でつくので書かなくてOK）

---

### ステップ 5: フロントエンドのコードを貼り付ける

1. エディタの左側にある「ファイル」の横の **「+」ボタン** をクリック
2. **「HTML」** を選ぶ
3. ファイル名を聞かれるので **`index`** と入力してEnter（`.html` は自動でつくので書かなくてOK）
4. 新しいファイルが開くので、最初から書いてある中身を **全部消す**
5. このリポジトリにある **`index.html`** ファイルの中身を **すべてコピー** して **貼り付ける**

> ここまでで、エディタの左側に **`main.gs`** と **`index.html`** の2つのファイルがあればOK！

---

### ステップ 6: APIキーを登録する

1. エディタの左側にある **歯車アイコン（プロジェクトの設定）** をクリック
2. 下にスクロールして **「スクリプト プロパティ」** を探す
3. **「スクリプト プロパティを追加」** をクリック
4. 2つの入力欄が出てくるので、こう入力する:
   - **プロパティ** の欄 → `GEMINI_API_KEY`
   - **値** の欄 → ステップ1でコピーしたAPIキーを貼り付ける
5. **「スクリプトプロパティを保存」** ボタンを押す

---

### ステップ 7: アプリを公開（デプロイ）する

1. エディタの右上にある **「デプロイ」** ボタンをクリック
2. **「新しいデプロイ」** をクリック
3. 出てきた画面の左上にある **歯車アイコン（種類の選択）** をクリック
4. **「ウェブアプリ」** を選ぶ
5. 以下の設定になっていることを確認する:
   - **実行するユーザー** → **「自分」**
   - **アクセスできるユーザー** → **「自分のみ」**
6. **「デプロイ」** ボタンを押す

---

### ステップ 8: アクセスを許可する

デプロイ直後に「承認が必要です」という画面が出ます。これは自分のスプレッドシートにデータを書き込む許可を求めているだけなので、安全です。

1. **「アクセスを承認」** をクリック
2. 自分の **Googleアカウントを選ぶ**
3. 「このアプリは確認されていません」と出たら → 左下の **「詳細」** をクリック
4. **「〇〇（安全ではないページ）に移動」** をクリック
5. **「許可」** をクリック

---

### ステップ 9: アプリを使う！

1. デプロイが完了すると **「ウェブアプリのURL」** が表示される
2. そのURLをクリックすると、アプリが開く
3. テキスト欄に質問を入力して「送信」を押せば、AIが答えてくれる

> **ブックマーク登録しておくと、次から簡単にアクセスできます！**

## 音声読み上げ（🔊ボタン）がうまく動かないとき

お使いの端末に対象言語の音声データが入っていないと、🔊ボタンを押しても音が出ないことがあります。
以下の手順で音声データを追加してください。

| 端末 | 手順 |
|---|---|
| **Windows** | 「設定」→「時刻と言語」→「言語と地域」→「言語の追加」から対象言語を追加 |
| **Android** | 「設定」→「システム」→「言語と入力」→「テキスト読み上げ」→ Googleテキスト読み上げエンジンの設定から音声データをダウンロード |
| **iPhone / iPad** | 「設定」→「アクセシビリティ」→「読み上げコンテンツ」→「声」から対象言語の音声をダウンロード |
| **Mac** | 「システム設定」→「アクセシビリティ」→「読み上げコンテンツ」→「システムの声」→「声を管理」から音声をダウンロード |

## 対応言語

| 言語 | 音声コード |
|---|---|
| 自動判定 | AIが自動で判定 |
| 英語 | `en-US` |
| 中国語 | `zh-CN` |
| 韓国語 | `ko-KR` |
| スペイン語 | `es-ES` |
| フランス語 | `fr-FR` |
| アラビア語 | `ar-SA` |

## コードを変更したあとの再デプロイ方法

コードを編集したあとは、もう一度デプロイし直す必要があります。

1. エディタ右上の **「デプロイ」** → **「デプロイを管理」** をクリック
2. 右上の **えんぴつアイコン（編集）** をクリック
3. **「バージョン」** を **「新バージョン」** に変更
4. **「デプロイ」** ボタンを押す

> URLは変わらないので、ブックマークはそのまま使えます。
