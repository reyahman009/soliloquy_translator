# 多言語語学学習サポートAI（スプレッドシート連携）

Google Apps Scriptを用いた多言語対応の語学学習チャットアプリの実装とセットアップ手順です。

## 1. アプリでできること（主な機能）

- **AIとの自然な会話・質問**: 英語、中国語、韓国語、スペイン語、フランス語、アラビア語など、学習中の言語に関する疑問（フレーズ、文法、ニュアンス）をAIに気軽に質問し、親切な解説をもらうことができます。
- **重要データの自動抽出**: あなたの質問と解説の中から、「外国語表現」「日本語訳」「例文」「ニュアンス」などの重要な構成要素をAIが自動で整理・抽出します。
- **スプレッドシートへの自動蓄積**: 抽出されたデータは、裏側で自動的にGoogleスプレッドシートのリストとして保存されていきます。あなた専用の「単語帳・フレーズ集」が質問するだけで勝手に出来上がります。
- **ネイティブ音声の読み上げ (TTS)**: 抽出されたフレーズや例文の横にある🔊ボタンを押すと、OSやブラウザの標準音声機能（Web Speech API）を利用して、対象言語のネイティブ発音を聴くことができます。端末に対応する言語の音声がない場合は、OS別の対応手順がエラーメッセージとして表示されます（詳しくは後述の「音声読み上げ (TTS) の対応方法」を参照）。
- **AIモデルの切替**: 用途や速度の好みに合わせて、以下の3つのGeminiモデルから選択できます。選択はlocalStorageに保存され、次回アクセス時にも引き継がれます。
  - 🪶 **Gemini 2.5 Flash Lite** — 高速・軽量（デフォルト）
  - ⚡ **Gemini 2.5 Flash** — バランス型
  - 💎 **Gemini 2.5 Pro** — 高精度
- **キーボードショートカット**: テキスト入力欄で `Ctrl + Enter` を押すと、メッセージを送信できます。
- **マルチデバイス対応**: スマホ、タブレット、PCのどのブラウザからでも見やすい画面設計で、スキマ時間にいつでもどこでも学習をスタートできます。

## 2. このアプリが活用できるシーン

- **第二外国語の学習サポート**
  英語のみならず、学習を始めたばかりの言語に対して「これってどう言うの？」と質問し、言語ごとの特有のニュアンスを理解します。
- **海外ドラマや映画のセリフの解読**
  字幕やスクリプトで気になったスラングの細かいニュアンスを質問して掘り下げ、自分だけのフレーズ集を作成します。
- **発音の確認（リスニング強化）**
  テキストを読むだけでなく、ネイティブ音声ボタンを活用して発音を繰り返し確認し、シャドーイングの練習に役立てます。
- **日々の「独り言（Soliloquy）」のお供に**
  日常のちょっとした疑問をいつでもスマホやPCから壁打ちして言語化します。

## 3. セットアップ手順

### 3-1. プロジェクトの作成
1. Googleドライブを開き、「新規」>「Google スプレッドシート」を作成します。
2. 作成したスプレッドシートのメニューから、「拡張機能」>「Apps Script」をクリックし、エディタを開きます。

### 3-2. ファイルの作成とコードの貼り付け
1. **バックエンド (main.gs)**
   - 新しく main.gs というスクリプトファイルを作成（またはデフォルトのファイルをリネーム）し、提供された内容を貼り付けます。
2. **フロントエンド (index.html)**
   - エディタ左側のファイル一覧にある「＋」ボタンをクリックし、「HTML」を選択します。
   - ファイル名を `index` (拡張子不要)とし、提供された index.html の内容を貼り付けます。

### 3-3. スクリプトプロパティへの設定（Gemini APIキーの登録）
1. GASエディタの左側メニュー（歯車アイコン）から「プロジェクトの設定」を開きます。
2. 「スクリプト プロパティ」のセクションまでスクロールし、「スクリプト プロパティを追加」をクリックします。
3. 以下の通り入力します：
   - プロパティ: `GEMINI_API_KEY`
   - 値: （取得したGemini APIキーを貼り付け）
4. 「スクリプトプロパティを保存」ボタンをクリックします。

### 3-4. デプロイ設定
1. エディタ右上の「デプロイ」>「新しいデプロイ」をクリックします。
2. 「種類の選択」の歯車アイコンをクリックし、「ウェブアプリ」を選択します。
3. 以下の設定がされていることを**必ず確認**してください：
   - 説明: 語学学習アシスタント（任意）
   - 実行するユーザー: **「自分」**
   - アクセスできるユーザー: **「自分のみ」**
4. 「デプロイ」ボタンをクリックします。
5. アクセス承認が求められた場合は、「アクセスを承認」>対象のGoogleアカウントを選択>「詳細」>「◯◯（安全ではないページ）に移動」>「許可」の順に進めます。
6. 発行された「ウェブアプリのURL」にアクセスして使用を開始します。

## 4. アプリの仕組み（動作フローとコード解説）

このアプリは、すべて作成者のGoogleアカウント内で完結する **完全無料・保守ゼロ** のアーキテクチャで構成されています。裏側で「どのファイル」の「どの処理」が動くかを順を追って解説します。

### ① UIの表示とユーザー入力 (フロントエンド: `index.html`)
- **画面描画**: アプリのURLにアクセスすると、バックエンドの `doGet(e)` 関数が呼ばれ、フロントエンドの画面を表示します。デザインにはTailwind CSSを利用し、モダンなUIを採用しています。
- **言語選択と記録**: 画面下部の言語プルダウン（自動判定 / 英語 / 中国語 / 韓国語 / スペイン語 / フランス語 / アラビア語）で対象言語を選択すると、`localStorage` に状態が記憶され、次回アクセス時も同じ言語が選ばれた状態から再開できます。
- **モデル選択と記録**: 言語プルダウンの横にあるモデル選択プルダウンで、利用するGeminiモデル（Flash Lite / Flash / Pro）を切り替えることができます。選択は `localStorage` に保存されます。
- **メッセージ送信**: 送信ボタンのクリック、または `Ctrl + Enter` でメッセージを送信します。テキストエリアは入力量に応じて自動で高さが調整されます。
- **バックエンド通信**: 非同期にバックエンドの `processChat` へメッセージ・選択した言語・選択したモデルの情報を送信します。

### ② Gemini APIへのリクエスト (バックエンド: `main.gs`)
- **プロンプト生成**: `main.gs` 内の `callGeminiApi` が、ユーザーの質問と「選択した対象言語」の情報をセットでプロンプト化します。
- **JSONスキーマの指定**: 単語の意味を解説させつつ、確実に抽出データ（表現、訳、例文、言語名）をJSON形式で整理して返すよう、構造化出力（`responseSchema`）機能でモデルに強制します。
- **API通信実行**: GASに組み込まれた `UrlFetchApp.fetch(...)` を使用し、フロントエンドで選択されたGeminiモデルへリクエストを送信します。

### ③ AIによる生成とデータ抽出 (Gemini API)
- Geminiは要求通り、以下の要素を持ったJSONを生成して返却します。
  1. `reply_message`: チャットUIに表示するための、自然な日本語での回答文
  2. `extracted_data`: スプレッドシート保存および画面UI向けの構造化データ（フレーズ、例文一覧、判定された言語など）

### ④ スプレッドシートへの自動保存 (バックエンド: `main.gs`)
- AIの生成完了後、`saveToSheet` 関数が呼び出され、タイムスタンプや対象言語のカラムを含むデータをシートの最終行に追記します。
- シートが空の場合は、初回保存時にヘッダー行（ID, Timestamp, Language, Phrase, Translation, Nuance/Context, Examples, Tags）が自動で作成されます。

### ⑤ 画面への回答表示と音声機能 (フロントエンド: `index.html`)
- **UIへの反映**: `extracted_data` のフレーズや例文は、AIの回答吹き出しの下部にフラッシュカード状のUIとして展開されます。
- **音声再生 (SpeechSynthesis)**: フレーズや例文には🔊再生ボタンが紐づけられており、クリック時にブラウザ標準のWeb Speech APIを通じて、指定された言語コードに紐づくネイティブ音声が再生されます。音声エンジンの検索は「完全一致 → アンダースコア表記 → 前方一致」の3段階フォールバックで実行されます。

## 5. 音声読み上げ (TTS) の対応方法

端末に該当言語の音声エンジンがインストールされていない場合、OS別の対応手順がエラーメッセージで案内されます（ユーザーのOSが自動判定され、該当するOSの手順がハイライト表示されます）。

| OS | 対応手順 |
|---|---|
| 🖥️ **Windows** | 「設定」→「時刻と言語」→「言語と地域」→「言語の追加」から対象言語を追加 |
| 📱 **Android** | 「設定」→「システム」→「言語と入力」→「テキスト読み上げ」→ Googleテキスト読み上げエンジンの設定から音声データをダウンロード |
| 🍎 **iOS / iPadOS** | 「設定」→「アクセシビリティ」→「読み上げコンテンツ」→「声」から対象言語の音声をダウンロード |
| 💻 **Mac** | 「システム設定」→「アクセシビリティ」→「読み上げコンテンツ」→「システムの声」→「声を管理」から音声をダウンロード |
